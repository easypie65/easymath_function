이차함수 동영상 촬영하기

⚠️ 촬영 시 주의사항

  • 프레임에 벗어나지 않게 높이와 거리 조절 필요
  • 공만 움직이고 주변 배경은 움직이지 않게 촬영
  • 충분한 조명 확보하기
  • 안전거리 유지하기

📸 완벽한 촬영을 위한 상세 가이드

🎯
촬영 각도 설정
  • 공의 궤적이 잘 보이는 측면에서 촬영
  • 던지는 사람의 옆쪽 45도 각도가 최적
  • 너무 가까우면 공이 화면 밖으로 나갈 수 있음
💡 팁: 미리 몇 번 던져보고 궤적을 파악한 후 촬영하세요!
📱
카메라 고정
  • 삼각대나 안정된 곳에 핸드폰을 고정
  • 손으로 들고 찍으면 화면이 흔들림
  • 책상, 의자 등을 활용해서 고정
💡 팁: 핸드폰 케이스나 클립을 활용하면 더 안정적!
⏱️
촬영 타이밍
  • 공이 던져지기 전부터 촬영 시작
  • 공이 완전히 착지할 때까지 촬영
  • 여유롭게 3-5초 더 찍어두기
💡 팁: 동영상은 나중에 편집할 수 있으니 넉넉하게!

Action Sequence Shot 앱 사용하기

📱 Action Sequence Shot 앱 완벽 사용법

앱스토어나 플레이스토어에서 "Action Sequence Shot" 검색하여 다운로드하세요!

①
동영상 불러오기
  • 앱 실행 → 화면의 "갤러리 아이콘" 터치
  • 방금 촬영한 스포츠 동영상을 선택
  • 동영상이 너무 길면 미리 편집해두는 것이 좋음
💡 동영상 길이는 10-30초 정도가 적당합니다
②
장면 선택하기
  • 화면 하단의 시간 바에서 시작점과 끝점을 드래그로 설정
  • 공이 던져지는 순간부터 착지까지만 선택
  • 불필요한 부분은 제외하고 핵심 궤적만 포함
💡 미리보기로 확인해가며 정확한 구간을 설정하세요
③
스샷 간격 선택
  • 슬라이더를 왼쪽으로 이동 (작은 숫자로 설정)
  • 간격이 작을수록 → 더 촘촘한 연속 사진이 만들어짐
  • 추천 간격: 0.1~0.2초
💡 너무 촘촘하면 사진이 복잡해지고, 너무 성기면 궤적이 끊어져 보입니다
④
연속사진 생성
  • 화면 중앙의 큰 카메라 버튼을 터치
  • 잠시 기다리면 처리가 완료됩니다 (약 10~30초)
  • 처리 중에는 다른 앱을 사용하지 마세요
⏳ 동영상이 길거나 간격이 작을수록 처리 시간이 더 걸립니다
⑤
이미지 저장
  • 우측 상단의 저장 버튼 (다운로드 아이콘) 터치
  • 갤러리에 연속사진이 자동으로 저장됩니다
  • 저장 완료되면 갤러리에서 바로 확인 가능
✅ 저장된 사진을 미리 확인해서 궤적이 잘 나왔는지 체크하세요!

사진 불러오기 및 설정

📷 GeoGebra에 사진 업로드하기

①
이미지 메뉴 찾기
  • 화면 상단 메뉴에서 "+" 버튼 클릭
  • 또는 "삽입" 메뉴 클릭
  • "이미지" 또는 "Image" 선택
②
연속사진 업로드
  • "파일 선택" 또는 "Browse" 클릭
  • 갤러리에서 Action Sequence Shot으로 만든 사진 선택
  • "열기" 또는 "Open" 클릭
③
사진 위치 조정
  • 사진을 드래그해서 적절한 위치로 이동
  • 모서리를 당겨서 크기 조절
  • 공의 궤적이 잘 보이는 크기로 설정
💡 너무 크면 그래프가 안 보이고, 너무 작으면 궤적이 안 보여요
④
투명도 설정
  • 사진을 우클릭 (또는 길게 터치)
  • "설정" 또는 "Properties" 선택
  • "색상" 탭에서 "불투명도" 조절
  • 추천: 50~70% 정도로 설정
💡 투명도를 조절해야 나중에 그래프가 사진 위에 잘 보입니다

✅ 사진 설정 완료 체크리스트

사진이 화면에 잘 보임
적당한 크기로 조절됨
투명도가 조절됨 (50-70%)
공의 궤적이 명확히 보임

이차함수 그래프 찾기

📐 이차함수 그래프 만들기

①
이차함수 입력하기

화면 왼쪽 입력창에 다음과 같이 입력:

y = a*x^2 + b*x + c
💡 입력하면 자동으로 a, b, c 슬라이더가 생성됩니다!
②
슬라이더로 값 조절하기
a 값 조절

포물선의 방향과 폭

음수: 아래볼록 / 양수: 위볼록
b 값 조절

좌우 이동

꼭짓점의 x좌표에 영향
c 값 조절

상하 이동

y절편 (x=0일 때 y값)
🎯 화면 왼쪽에 나타나는 슬라이더를 드래그하여 값을 조절하세요
③
그래프와 사진 맞추기
  • 공의 궤적과 그래프 곡선이 최대한 겹치도록 조절
  • 특히 최고점(꼭짓점)이 잘 맞는지 확인
  • 시작점과 끝점도 비슷한 위치에 있는지 확인
  • 전체적인 곡선의 모양이 비슷한지 확인
💡 완벽하게 맞지 않아도 괜찮아요! 최대한 비슷하게 만드는 것이 목표입니다

⚠️ 그래프를 찾기 어려울 때

  • 수기로 바꿔서 그래프를 조정해보세요
  • 슬라이더 대신 직접 숫자를 입력할 수도 있습니다
  • 완벽하지 않아도 괜찮습니다. 최대한 비슷하게 맞춰보는 것이 중요!

그래프 식 표시 및 저장

📋 최종 결과물 완성하기

①
그래프 식 화면에 표시하기
  • 그래프 곡선을 우클릭 (또는 길게 터치)
  • "설정" 또는 "Properties" 선택
  • "라벨 표시" 또는 "Show Label" 체크
  • 화면에 y = ax² + bx + c 식이 나타남
💡 식이 잘 보이지 않으면 글씨 크기나 위치를 조정하세요
②
이미지로 저장하기

두 가지 방법 중 편한 방법 선택:

방법 1: 메뉴 이용
  • 화면 상단 "내보내기" 클릭
  • "이미지로 다운로드" 선택
  • PNG 형식으로 저장
방법 2: 우클릭 이용
  • 빈 공간에서 우클릭
  • "그래픽 뷰 내보내기"
  • 파일명 입력 후 저장
📸 파일명 추천: "학번_이름_스포츠종목.png" (예: 0315_김수학_농구.png)

✅ 최종 확인 체크리스트

연속사진과 그래프가 모두 선명하게 보임
이차함수 식 (y = ax² + bx + c)이 화면에 표시됨
그래프와 공의 궤적이 거의 일치함
파일명이 적절하게 설정됨
조원 정보가 준비됨
